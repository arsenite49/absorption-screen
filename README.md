# Absorption Screen

A browser-based accumulation screener for NSE smallcaps: ranks stocks by relative
strength on index down-days combined with delivery flow vs each stock's own baseline.

**Use it:** https://arsenite49.github.io/absorption-screen/

- Click **"Preview with synthetic data"** to explore instantly — no data needed.
- For real screening, load official NSE `sec_bhavdata_full_*.csv` files (plus optional
  index history, constituents, and ASM/GSM / bulk-deal CSVs). Everything runs locally
  in your browser; nothing is uploaded anywhere.

Educational use only. Not investment advice; no buy/sell/hold calls.
